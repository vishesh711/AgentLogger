GROQ API KEY SETUP INSTRUCTIONS

1. Create a .env file in the project root with the following content:

```
# Project settings
PROJECT_NAME=AgentLogger API
PROJECT_DESCRIPTION=AI-powered debugging API service
VERSION=0.1.0

# API settings
API_V1_STR=/api/v1

# CORS settings
CORS_ORIGINS=http://localhost:3000,http://localhost:8000

# Database settings
POSTGRES_SERVER=localhost
POSTGRES_USER=postgres
POSTGRES_PASSWORD=postgres
POSTGRES_DB=agentlogger

# Redis settings (optional)
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_PASSWORD=
USE_REDIS=false

# Security settings
SECRET_KEY=your-secret-key-here

# LLM settings
GROQ_API_KEY=your-groq-api-key-here
GROQ_MODEL=llama3-70b-8192

# GitHub integration (optional)
GITHUB_ACCESS_TOKEN=

# Sandbox execution
USE_DOCKER_SANDBOX=true
EXECUTION_TIMEOUT=30

# Rate limiting
RATE_LIMIT_PER_MINUTE=60
```

2. Make sure to keep this file secure and never commit it to version control.
3. The .env file is already included in .gitignore to prevent accidental commits.

IMPORTANT: Your Groq API key is a sensitive credential. Never share it publicly or commit it to GitHub.
